## Programs Used
- arandr
- xrandr
- feh
- srandrd (aur) : runs a script when a display is connected / disconnected
- zsh
- prezto (for zsh)
- polybar
- bspwm
- sxhkd
- xscreensaver (aur)
- backlight (baskerville on github)
- pactl (volume control)
- neovim
- rofi
- firefox
- lightdm
- compton

## Fonts Used
- Wuncon Siji
- SF Pro Display
- Source Code Pro
- Maboroshi no Nijimi Mincho (polybar japanese characters)

## Icons
- Moka (for rofi)
